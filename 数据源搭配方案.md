## 一、数据源现状

| 数据源 | 实时行情 | 日/周/月K | 分钟线 | 稳定性 | 速度 |
|--------|----------|-----------|--------|--------|------|
| 腾讯 qt.gtimg.cn | ✓ | ✗ | ✗ | 高 | 0.2s |
| 腾讯 web.ifzq | ✗ | ✓ | ✗ bad | 高 | 0.3s |
| 东方财富 | ✗ | ✓ | ✓ | 中 | 0.2s |
| Tushare | ✗ | ✓ | ✗ | 高 | 0.5s |

**关键发现：**
- 腾讯分钟线接口已废弃（bad params）
- 东方财富需要JSONP调用，浏览器中可用
- 腾讯实时行情是唯一可用的实时数据源

## 二、推荐搭配方案

### 方案A：稳定优先（推荐）

```
实时行情 → 腾讯 qt.gtimg.cn
日/周/月 → Tushare → 腾讯 web.ifzq → 东方财富
分钟线   → 东方财富（JSONP）
```

**优势：**
- Tushare日线数据最稳定
- 腾讯实时行情响应快
- 东方财富兜底分钟线

**代码实现：**
```javascript
// 实时行情
async function fetchRealtime(code) {
    const resp = await fetch(`https://qt.gtimg.cn/q=${prefix}${code}`);
    const text = await resp.text();
    // GBK解码...
}

// K线数据
async function fetchKline(code, period) {
    if (['5','30','60'].includes(period)) {
        // 分钟线：只用东方财富
        return fetchEastMoney(code, period);
    } else {
        // 日/周/月：Tushare → 腾讯
        return fetchTushare(code, period)
            .catch(() => fetchTencent(code, period));
    }
}
```

### 方案B：速度优先

```
实时行情 → 腾讯 qt.gtimg.cn
日/周/月 → 腾讯 web.ifzq
分钟线   → 东方财富（JSONP）
```

**优势：**
- 腾讯响应快
- 减少数据源切换

**劣势：**

## 三、各接口调用方式

### 3.1 实时行情（新增）

```javascript
// 腾讯实时行情
const REALTIME_API = 'https://qt.gtimg.cn/q=';

// 调用
const prefix = getMarketPrefix(code);  // sh 或 sz
const url = `${REALTIME_API}${prefix}${code}`;
const resp = await fetch(url);
const buffer = await resp.arrayBuffer();
const text = new TextDecoder('gbk').decode(buffer);

// 解析
const fields = text.split('~');
const quote = {
    name: fields[1],
    code: fields[2],
    price: parseFloat(fields[3]),
    close: parseFloat(fields[4]),  // 昨收
    open: parseFloat(fields[5]),
    volume: parseInt(fields[6]),   // 成交量(手)
    high: parseFloat(fields[33]),
    low: parseFloat(fields[34]),
    change: parseFloat(fields[31]),
    changePct: parseFloat(fields[32]),
    amount: parseFloat(fields[37]),  // 成交额(万)
};
```

### 3.2 日K线

```javascript
// 腾讯K线
const url = `https://web.ifzq.gtimg.cn/appstock/app/kline/kline?param=${prefix}${code},day,,,200`;

// 返回：[日期, 开盘, 收盘, 最高, 最低, 成交量]
```

### 3.3 分钟线

```javascript
// 东方财富（JSONP方式，浏览器中使用）
function fetchEastMoney(code, period) {
    return new Promise((resolve, reject) => {
        const cb = 'cb_' + Date.now();
        window[cb] = (data) => {
            // 处理数据
            resolve(data.data.klines);
        };
        const script = document.createElement('script');
        script.src = `https://push2his.eastmoney.com/api/qt/stock/kline/get?...&cb=${cb}`;
        document.head.appendChild(script);
    });
}
```

```

   ↓

```

```javascript
        fetchRealtime(currentCode).then(updateQuoteDisplay);
```

## 六、注意事项

1. **GBK编码**：实时行情接口返回GBK，需要用 TextDecoder 解码
2. **CORS**：qt.gtimg.cn 可能有CORS限制，需要代理或JSONP
3. **频率限制**：避免过于频繁请求，建议≥5秒间隔
4. **交易时间**：仅在 9:30-11:30, 13:00-15:00 期间自动刷新
