## 三、自动启动配置

### 3.1 Tushare 后端
- **启动脚本**: `~/Desktop/41mbp 共享/output_html/tushare_backend.py`
- **服务端口**: `localhost:5000`
- **launchd 配置**: `~/Library/LaunchAgents/com.tushare.backend.plist`
- **状态**: ✓ 已配置开机自启

### 3.2 管理命令
```bash
# 查看状态
launchctl list | grep tushare

# 手动启动
launchctl start com.tushare.backend

# 手动停止
launchctl stop com.tushare.backend

# 卸载自启
launchctl unload ~/Library/LaunchAgents/com.tushare.backend.plist

# 重新加载
launchctl unload ~/Library/LaunchAgents/com.tushare.backend.plist
launchctl load ~/Library/LaunchAgents/com.tushare.backend.plist
```

## 四、页面数据流

```
┌─────────────────────────────────────────────────────────────┐
|  digdog.html (原 stock.html) v0.0.1   |
├─────────────────────────────────────────────────────────────┤
│  1. 用户查询股票                                            │
│     ↓                                                       │
│  2. 并行请求:                                               │
│     ├─ 实时行情: qt.gtimg.cn (快速显示价格)                 │
│     └─ K线数据: Tushare → 腾讯 (获取历史数据)              │
│     ↓                                                       │
│  3. 渲染图表:                                               │
│     ├─ 主图: K线 + GMMA                                    │
│     ├─ 副图1: MACD/RSI/KDJ/布林带/VOL                      │
│     ├─ 副图2: 成交额                                       │
│     └─ 副图3: D趋势                                        │
│     ↓                                                       │
│  4. 自动刷新: 每10秒刷新实时行情（仅交易时间）              │
└─────────────────────────────────────────────────────────────┘
```
