## 二、成交额量柱

### 2.1 是什么

每一根柱子代表当日的**成交金额**（单位：亿元）。

### 2.2 颜色含义

| 颜色 | 含义 |
|------|------|
| 🟢 绿色柱 | 当日**上涨**（收盘 ≥ 开盘）|
| 🔴 红色柱 | 当日**下跌**（收盘 < 开盘）|

### 2.3 柱子高度

- **柱子越高** → 成交额越大 → 交易活跃
- **柱子越低** → 成交额越小 → 交易清淡

### 2.4 看盘要点

```
放量上涨（绿柱高）= 多头积极，趋势可能延续
放量下跌（红柱高）= 空头抛压，注意风险
缩量横盘（柱子低）= 观望情绪，等待方向
```

## 三、Delta 效率图

### 3.1 组成部分

```
┌─────────────────────────────────────────┐
│                      ╱╲  ← 蓝色折线：累积 Delta
│                    ╱    ╲
│                  ╱        ╲
│  ┃ ┃   ┃  ┃   ┃  ┃  ┃  ┃  ← 彩色柱子：效率
│  绿  黄  红  绿  黄  绿  红
└─────────────────────────────────────────┘
```

### 3.2 蓝色折线：累积 Delta

**含义：** 多空力量的累计对比

| 形态 | 含义 |
|------|------|
| 折线**上升** | 多方持续占优 |
| 折线**下降** | 空方持续占优 |
| 折线**走平** | 多空平衡 |
| 折线**拐头** | 趋势可能反转 |

### 3.3 彩色柱子：Delta 效率

**含义：** 每一块钱撬动了多少价格波动

| 颜色 | 含义 | 解读 |
|------|------|------|
| 🟢 绿色柱 | 高效率 | 资金使用效率高，花小钱办大事 |
| 🟡 黄色柱 | 中等效率 | 正常水平 |
| 🔴 红色柱 | 低效率 | 资金使用效率低，花大钱办小事 |

## 四、量柱 + 折线 + 价格 联合判断

### 4.1 四种典型场景

#### 场景 1：量柱高 + 绿色效率柱 → 真实上涨 ✅

```
价格：上涨
成交额：放大
效率：高（绿色）

解读：资金积极入场，上涨有支撑
操作：可跟随做多
```

#### 场景 2：量柱高 + 红色效率柱 → 危险信号 ⚠️

```
价格：震荡或小幅涨跌
成交额：放大
效率：低（红色）

解读：大量资金换手但价格不动，可能是出货
操作：谨慎观望，不宜追高
```

#### 场景 3：量柱低 + 绿色效率柱 → 不可持续 ⚠️

```
价格：波动较大
成交额：萎缩
效率：高（绿色）

解读：少量资金撬动价格，缺乏持续性
操作：等待放量确认
```

#### 场景 4：量柱低 + 黄色效率柱 → 缩量横盘 📊

```
价格：窄幅震荡
成交额：萎缩
效率：中等

解读：多空双方观望，等待方向选择
操作：保持耐心，等突破
```

### 4.2 快速判断表

| 成交额 | 效率 | 价格 | 信号 | 建议 |
|--------|------|------|------|------|
| 大 | 高（绿） | 上涨 | 强势 | 跟随 |
| 大 | 低（红） | 横盘 | 换手 | 观望 |
| 小 | 高（绿） | 涨/跌 | 虚弱 | 等待 |
| 小 | 中（黄） | 横盘 | 整理 | 耐心 |

## 五、常见形态解读

### 5.1 折线 + 价格背离

**顶背离（卖出信号）：**
```
价格：创新高 ↗
折线：未创新高 →

含义：上涨动力减弱
建议：注意止盈，防范回调
```

**底背离（买入信号）：**
```
价格：创新低 ↘
折线：未创新低 →

含义：下跌动力减弱
建议：关注反弹机会
```

### 5.2 效率柱突变

**突然出现高绿柱：**
```
含义：资金突然活跃
关注：是否有消息面刺激
```

**连续出现高红柱：**
```
含义：持续低效换手
警惕：可能在出货
```

## 七、注意事项

1. **指标仅供参考**：任何指标都不是万能的，需结合多种分析方法

2. **关注趋势而非单日**：单日的效率波动可能有噪音，关注连续趋势

3. **大盘 vs 个股**：
   - 大盘股：指标更稳定可靠
   - 小盘股：指标可能被少量资金扭曲

4. **配合其他指标**：
   - GMMA 均线判断趋势方向
   - 价格行为判断入场时机
   - Delta 效率验证信号强度

> 三者结合，判断资金的真实意图。
