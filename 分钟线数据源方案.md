## 解决方案

### 方案1：使用日K代替（推荐）

对于技术分析，日K数据通常足够：
- Al Brooks 价格行为学主要基于日K
- GMMA、MACD等指标在日K上更稳定
- 分钟线噪音较多

**操作：** 使用日K/周K/月K进行分析

---

### 方案2：升级Tushare积分

Tushare分钟线需要2000积分：
- 访问：https://tushare.pro
- 查看当前积分
- 积分获取方式：捐赠、数据贡献等

**后端代码添加：**
```python
@app.route('/api/stock/minutes', methods=['GET'])
def get_minutes_data():
    """获取分钟线数据"""
    try:
        code = request.args.get('code', '000001')
        freq = request.args.get('freq', '5')  # 1/5/15/30/60
        limit = int(request.args.get('limit', 200))
        
        ts_code = f"{code}.{'SZ' if code.startswith('0') or code.startswith('3') else 'SH'}"
        
        # 需要2000积分
        df = pro.stk_mins(ts_code=ts_code, freq=freq, 
                          start_date='20240101', end_date=datetime.now().strftime('%Y%m%d'))
        
        if df.empty:
            return jsonify({'success': False, 'error': '没有数据'}), 404
        
        df = df.tail(limit)
        
        data = []
        for _, row in df.iterrows():
            data.append({
                'time': int(datetime.strptime(str(row['trade_time']), '%Y%m%d%H%M%S').timestamp()),
                'open': float(row['open']),
                'high': float(row['high']),
                'low': float(row['low']),
                'close': float(row['close']),
                'volume': float(row['vol']) if 'vol' in row else 0
            })
        
        return jsonify({'success': True, 'data': data})
    except Exception as e:
        return jsonify({'success': False, 'error': str(e)}), 500
```

### 方案4：AKShare（免费替代）

AKShare 是免费的金融数据接口库：

```python
import akshare as ak

# 获取分钟线
df = ak.stock_zh_a_hist_min_em(symbol="600519", period="5", 
                                start_date="2024-01-01", end_date="2024-04-14")
```

**优点：**
- 免费
- 支持分钟线
- 数据源是东方财富

**缺点：**
- 可能不稳定
- 需要安装akshare库
