## 一、Delta 概念定义

### 1.1 什么是 Delta

Delta 是衡量某一时间周期内**买卖双方力量对比**的指标。在 A 股市场中，由于无法直接获取逐笔成交数据（主动买/主动卖），我们采用**价格行为推算法**来估算 Delta 值。

### 1.2 计算公式

```
价格振幅 = 最高价 - 最低价
价格方向 = 收盘价 >= 开盘价 ? 1 : -1
价格推动力 = |收盘价 - 开盘价| / 价格振幅

Delta = 价格推动力 × 成交量 × 价格方向
```

**公式解读：**
- `价格推动力`：表示 K 线实体占整个振幅的比例，反映多空双方的实际控制力
- `价格方向`：阳线为 +1，阴线为 -1
- `Delta > 0`：多方主导
- `Delta < 0`：空方主导

### 1.3 特殊情况处理

| 情况 | 处理方式 |
|------|----------|
| 一字涨停/跌停 | 价格振幅为 0，设为 0.01 避免除零 |
| 十字星 | 价格推动力接近 0，Delta 接近 0 |
| 长上下影线 | 实体占比小，Delta 值较小 |

## 二、Delta 效率指标

### 2.1 什么是 Delta 效率

Delta 效率衡量的是**单位资金撬动的价格波动能力**，反映了资金的使用效率。

### 2.2 计算公式

```
成交额（元）= 成交量 × 平均价格
成交额（万元）= 成交额 / 10000

Delta效率 = |Delta| / 成交额（万元）
```

**公式解读：**
- 效率高：用较少的资金撬动了较大的价格波动
- 效率低：用了大量资金但价格波动不大

### 2.3 效率评级

| 评级 | 条件 | 含义 |
|------|------|------|
| 高效率 ✅ | 效率 > 平均效率 × 1.5 | 资金使用效率高，趋势真实 |
| 中等 | 其他 | 正常水平 |
| 低效率 ⚠️ | 效率 < 平均效率 × 0.5 | 资金消耗大，可能换手出货 |

## 三、量价关系分析框架

### 3.1 四象限分析模型

根据 **成交额大小** 和 **Delta 效率高低**，形成四个象限：

```
                    Delta 大（效率高）
                         │
        ┌────────────────┼────────────────┐
        │                │                │
        │   缩量上涨     │    真实趋势    │
        │   （观望）     │    （跟随）    │
        │                │                │
成交额 ─┼────────────────┼────────────────┤
  小    │                │                │
        │   缩量横盘     │    小资金撬动  │
        │   （观望）     │    （警惕）    │
        │                │                │
        └────────────────┼────────────────┘
                         │
                    Delta 小（效率低）
                         成交额大
```

### 3.2 四种典型场景

#### 场景 1：成交额大 + Delta 大 → 真实趋势 ✅

**特征：**
- 大量资金入场
- 价格波动明显
- 资金使用效率高

**解读：**
- 主力资金积极运作
- 趋势具有持续性
- 可以跟随操作

**操作建议：**
- 顺势而为
- 设置合理止损

---

#### 场景 2：成交额大 + Delta 小 → 高位换手危险 ⚠️

**特征：**
- 大量资金流入流出
- 价格波动有限
- 资金使用效率低

**解读：**
- 可能是高位换手
- 主力可能在出货
- 多空分歧严重

**操作建议：**
- 谨慎观望
- 不宜追高
- 注意风险控制

---

#### 场景 3：成交额小 + Delta 大 → 小资金撬动 ⚠️

**特征：**
- 少量资金推动较大波动
- 资金使用效率看似高
- 流动性不足

**解读：**
- 可能是小资金试探
- 市场参与度低
- 趋势不可持续

**操作建议：**
- 不宜重仓
- 等待放量确认
- 注意假突破

---

#### 场景 4：成交额小 + Delta 小 → 缩量横盘 📊

**特征：**
- 成交清淡
- 价格波动小
- 市场观望情绪浓

**解读：**
- 多空双方均无动作
- 可能是变盘前兆
- 需要等待方向选择

**操作建议：**
- 保持观望
- 等待放量突破
- 关注消息面

## 四、累积 Delta 分析

### 4.1 累积 Delta 定义

```
累积Delta = Σ Delta（从起始日期到当前日期）
```

### 4.2 累积 Delta 的意义

| 指标 | 含义 |
|------|------|
| 累积 Delta > 0 且持续上升 | 多方长期主导，趋势向好 |
| 累积 Delta < 0 且持续下降 | 空方长期主导，趋势向弱 |
| 累积 Delta 走平 | 多空平衡，震荡行情 |
| 累积 Delta 拐头 | 趋势可能反转 |

### 4.3 累积 Delta 与价格背离

**顶背离：**
- 价格创新高
- 累积 Delta 未创新高
- 预示：上涨动力减弱，可能见顶

**底背离：**
- 价格创新低
- 累积 Delta 未创新低
- 预示：下跌动力减弱，可能见底

## 六、实战应用要点

### 6.1 操作流程

```
1. 观察累积 Delta 方向 → 判断大趋势
2. 分析当日 Delta 效率 → 判断资金意图
3. 结合量价四象限 → 确定操作策略
4. 配合价格行为 → 寻找入场时机
```

### 6.2 风险控制

| 情况 | 风险等级 | 建议 |
|------|----------|------|
| 成交额大 + Delta 效率低 | 高 | 可能出货，减仓 |
| 连续多日 Delta 效率下降 | 中 | 趋势减弱，注意止盈 |
| Delta 与价格严重背离 | 高 | 反转信号，谨慎 |
| 缩量 Delta 放大 | 中 | 不可持续，观望 |

### 6.3 注意事项

1. **Delta 是估算值**：由于 A 股无法获取逐笔成交数据，Delta 是基于价格行为推算的，存在一定误差

2. **结合其他指标**：Delta 不宜单独使用，应配合：
   - GMMA（顾比均线）判断趋势
   - 价格行为（Al Brooks）判断入场点
   - MACD/RSI 等辅助验证

3. **不同周期差异**：
   - 日线 Delta：反映中期趋势
   - 周线 Delta：反映长期趋势
   - 分钟线 Delta：反映短期波动

4. **个股差异**：
   - 大盘股：Delta 更能反映真实资金流向
   - 小盘股：Delta 可能被少量资金扭曲
